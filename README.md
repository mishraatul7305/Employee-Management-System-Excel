# Employee Management System Dashboard (Excel)

## Project Overview
This project demonstrates an Employee Management System created in Microsoft Excel. It includes employee data analysis, dashboards, charts, and Excel functions.

## Features
- Employee database (500 records)
- Salary Analysis
- Highest Salary
- Lowest Salary
- Average Salary
- Total Salary
- Employee Count
- Sort and Filter
- Conditional Formatting
- Column Chart
- Doughnut/Pie Chart
- Interactive Dashboard
## Excel Skills Used
- SUM
- AVERAGE
- MAX
- MIN
- COUNT
- IF
- COUNTIF
- Sorting
- Filtering
- Charts
- Cell Formatting
